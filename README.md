# WindhorseTour — 2026 Website Re-theme (RFP)

Public bidding materials for the 2026 WordPress block-theme re-theme.


## What is public vs NDA-gated

| Access | Materials |
|--------|-----------|
| **Public (this repo)** | RFP, reference docs, Q&A log, NDA template, commercial template, workflow |
| **After executed NDA** | Read-only GitHub (finalists), Figma collaborator access, staging — see [03-legal/how-to-sign-nda.md](03-legal/how-to-sign-nda.md) |
| **After contract** | Full repo, environments, WindhorseTour-provisioned GitHub accounts |

Questions: **Slack** once WindhorseTour invites you to bid. No email.
